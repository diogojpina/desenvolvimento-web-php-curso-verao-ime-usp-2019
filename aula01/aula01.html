Protocolo HTTP

  • Mensagem: Cabeçalho + Corpo
  • Método: GET, POST, PUT, DELETE, HEAD, TRACE, OPTIONS, CONNECT
  • Parâmetros, /index.php?a=1&b=2
  • Cookies
  • Código de Retorno
    • 1xx: Informação
    • 2xx: Sucesso
    • 3xx: Redirecionamento
    • 4xx: Erro do cliente
    • 5xx: Erro do servidor

Variáveis

  • Inicia com o caracter $
  • Não comece com números.
  • Não utilize espaços em branco.
  • Não utlize caracters especiais.
  • O nome deve ser significativos e curtos.
  • Letras minúsculas e palavras separadas por "_" ou camel case.
  • Case sensitive

Exemplos


$num1 = 2;
$num2 = 3;
$num3 = "2";

($num1 == $num3) // true
($num1 === $num3) // false

($num1 != $num2) // true

($num1 > $num2) // false
($num1 < $num2) // true
($num1 >= $num2) // false
($num1 <= $num2) // true

Operadores Lógicos

Operador Descrição
($a && $b) E: Verdadeiro se $a e $b forem verdadeiros
($a || $b) OU: Verdadeiro se $a ou $b forem verdadeiros
($a and $b) E: Verdadeiro se $a e $b forem verdadeiros
($a or $b) OU: Verdadeiro se $a ou $b forem verdadeiros
($a xor $b) XOR: Verdadeiro se $a ou $b forem verdadeiros exclusivamente
! $a NOT: Verdadeiro se $a for falso

Estrutura de Controle - IF


$a = 5;

if ($a == 5) {
	echo "$a é igual a 5";
}
else if ($a = 7) {
	echo "$a é igual a 5";
}
else {
	echo "Nenhuma das condições anteriores";
}

Estrutura de Controle - WHILE


$i = 1;
while ($i <= 5) {
	echo $i;

	$i++;
}
//12345

Estrutura de Controle - FOR/FOREACH/


for ($i=1; $i <= 5; $i++) {
	echo $i;
}
//12345

$vetor = array("Pera", "Maçã", "Uva");
for ($i=0; $i < count($vetor); $i++) {
	echo $vetor[$i];
}
//PeraMaçãUva

foreach ($vetor as $chave => $valor) {
	echo "$chave->$valor ";
}
//0->Pera 1->Maçã 2->Uva
